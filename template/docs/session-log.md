# Session Log — <project-name>

Record of significant work sessions. Updated by the orchestrator at the end
of each session that makes meaningful changes.

Keeping this in a separate file rather than AGENTS.md prevents the system
prompt from growing unboundedly with historical entries.

---

## Format

| Date | Summary |
|------|---------|
| YYYY-MM-DD | One-line summary of what was done |
